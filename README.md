## ðŸ“š Contents

### **HTML Topics** (html_noted_codes.html)
Covers 15 topics including:
- Document structure, headings, paragraphs
- Text formatting, lists, links
- Tables, forms, semantic elements
- Div containers and navigation menus

### **CSS Topics** (Basics/css_noted_codes.css)
Covers 17 topics including:
- CSS selectors, colors, fonts
- Box model, backgrounds, positioning
- Flexbox and Grid layouts
- Pseudo-classes, pseudo-elements
- Transitions, animations, media queries
- CSS variables and mobile responsive design

### **JavaScript Topics** (Basics/javascript_noted_codes.js)
Covers 20 topics including:
- Variables, operators, control flow
- Functions, arrays, strings, objects
- Regular expressions, loops, switch statements
- ES6 classes, prototypes, destructuring
- Events, scope, hoisting
- REST API calls, JSON handling, async/await

### **Machine Problems** (Machine Problem Samples/)
Interactive examples demonstrating:
- String manipulation (replaceAll)
- Character counting
- Live DOM manipulation
