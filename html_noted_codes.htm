HTML Topics 1-15 - Noted Codes

Basics Tags Paragraphs Line Breaks Headings Formatting Lists Links Navigation Comments Semantics Tables Horizontal Line Forms Div

1. HTML Basics

Note: Every HTML page starts with <!DOCTYPE html> and has <html>, <head>, <body> tags.
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <p>Hello World!</p>
  </body>
</html>

2. HTML Tags & Elements

Note: Tags define elements. Some are block-level (p, div), some inline (a, span).
<h1>Heading</h1>
<p>Paragraph</p>
<a href="https://example.com">Link</a>
<img src="image.jpg" alt="Example">

3. HTML Paragraphs

Use <p> to separate text into paragraphs. Adds vertical spacing automatically.
<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

4. HTML Line Breaks

Use <br> to break lines without starting a new paragraph. Self-closing tag.
<p>Line 1<br>Line 2</p>

5. HTML Headings

Six levels: <h1> (most important) to <h6> (least important). Use sequentially.
<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>

6. HTML Text Formatting

Bold, italic, underline, strikethrough, superscript, subscript tags are used to emphasize text.
<p>This is <b>bold</b> text.</p>
<p>This is <i>italic</i> text.</p>
<p>This is <u>underlined</u> text.</p>
<p>This is <s>strikethrough</s> text.</p>
<p>E = mc<sup>2</sup></p>
<p>H<sub>2</sub>O</p>

7. HTML Lists

Use <ul> for unordered, <ol> for ordered lists. Items inside <li>.
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

8. HTML Links

Anchor tags <a> create links. Use href for URL or #id for internal sections.
<a href="https://example.com">Visit Site</a>
<a href="#topic3">Go to Paragraphs</a>

9. HTML Navigation

Use <nav> with lists and links to build menus.
<nav>
  <ul>
    <li><a href="#topic1">Home</a></li>
    <li><a href="#topic5">Headings</a></li>
  </ul>
</nav>

10. HTML Comments

Comments do not appear in browser. Useful for notes or temporarily hiding code.
<!-- This is a comment -->
<!-- <p>Hidden paragraph</p> -->

11. HTML Semantics

Semantic tags convey meaning & improve accessibility and SEO.
<header>Header</header>
<main>Main content</main>
<section>Section</section>
<article>Article</article>
<aside>Sidebar</aside>
<footer>Footer</footer>

12. HTML Tables

Use <table>, <tr>, <td>, <th>, <thead>, <tbody> to structure data.
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>

13. HTML Horizontal Line

Use <hr> to visually separate content sections.
<p>Above</p>
<hr>
<p>Below</p>

14. HTML Forms

Forms collect user input using inputs, textarea, select, radio, checkbox, and submit buttons.
<form action="/submit" method="post">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username">
  <input type="submit" value="Send Data">
</form>

15. HTML Div

<div> is a generic container for grouping elements. No semantic meaning.
<div>
  <h2>Title</h2>
  <p>Paragraph</p>
</div>
